Course Overview

This Radical Solutions Architecture Capstone Project is a practical and comprehensive project designed to test and enhance your knowledge and skills in cloud architecture.

Through hands-on labs and simulations, you will engage in tasks such as:

  • Cloning a web application repository
  • Integrating APIs
  • Creating VM instances using Terraform
  • Monitoring them with Cloud monitoring
  • Deploying applications on the cloud
  • Triggering Continuous Delivery pipelines
  • Enhancing web application security

Once you have successfully completed this capstone project, you will have confirmation of your expertise and practical proficiency, which you can highlight on your professional profiles and in your resume. Plus, you will have valuable hands-on examples of your skills that you can talk about in interviews.
